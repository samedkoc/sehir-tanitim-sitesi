# Şehir Tanıtım Sitesi Şablonu / City Promotion Website Template 🗺️✈️

### [Türkçe]
Modern web standartlarına uygun, tamamen duyarlı (responsive) ve dinamik bir şehir tanıtım sitesi şablonudur. Temiz, modüler ve anlaşılır kod yapısı sayesinde **içerikler, metinler, menüler ve görseller başka bir şehre göre kolaylıkla değiştirilebilir ve özelleştirilebilir.** Proje, şu an halihazırda Adana şehri verileriyle önizleme olarak doldurulmuştur.

### [English]
A modern, fully responsive city promotion website template. Thanks to its clean and modular code architecture, **all texts, images, navigation menus, and sections can be easily customized and modified for any other city.** The project is currently pre-filled with Adana city data as a live preview example.

## ⚙️ Kolay Değiştirilebilirlik ve Esneklik / Easy Customization

*   **Modüler Yapı:** HTML5 ve CSS3 yapısı tamamen yorum satırlarıyla desteklenmiş olup, yeni sayfalar eklemek veya mevcut içerikleri değiştirmek son derece basittir.
*   **Kolay Resim Yönetimi:** `img` klasöründeki görselleri değiştirerek veya URL bağlantılarını güncelleyerek siteyi dakikalar içinde bambaşka bir şehre uyarlayabilirsiniz.
*   **Hızlı CSS Özelleştirmesi:** Renk paletleri ve yazı tipleri (fonts) merkezi bir yapıda olduğu için kurumsal kimliğe göre kolayca manipüle edilebilir.
